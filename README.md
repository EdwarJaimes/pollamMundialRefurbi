# 🏆 Polla Refurbi 2026

¡Bienvenido a **Polla Refurbi**, la aplicación definitiva para organizar tu polla (quiniela) futbolística con la familia y amigos! 

Esta aplicación ha sido diseñada para ofrecer una experiencia fluida, visualmente atractiva y fácil de gestionar para todos los participantes.

## 🚀 Características Principales

- **📱 Experiencia Mobile-First**: Diseñada específicamente para ser usada desde el celular.
- **⚽ Predicciones Inteligentes**: Sistema de marcadores (+/-) que evita el uso del teclado y errores de dedo.
- **🥇 Ranking en Tiempo Real**: Tabla clasificatoria dinámica con medallas para el podio.
- **🛡️ Panel de Administración Seguro**: Acceso protegido para ingresar resultados oficiales y repartir puntos automáticamente.
- **✨ Diseño Premium**: Estética moderna con efectos de cristal (Glassmorphism), animaciones y modo oscuro.
- **🌐 Multilingüe**: Nombres de selecciones traducidos al español para una mejor experiencia local.

## 📏 Reglas de Puntuación

- **Acierto Exacto (+3 pts)**: El usuario acierta el marcador final idéntico.
- **Acierto de Ganador (+1 pt)**: El usuario acierta quién gana o si hay empate, pero no los goles exactos.
- **Fallo Total (0 pts)**: No se acierta el resultado ni el ganador.

## 🛠️ Tecnologías Usadas

- **Frontend**: HTML5, Vanilla CSS3, Javascript (ES6+).
- **Backend**: Firebase Cloud Firestore (Base de datos en tiempo real).
- **Estilos**: Google Fonts (Outfit), Custom CSS animations.
